# RHN Protocol

Personal 16-week workout tracker — built for one user, one program.

Static HTML / CSS / JS. No backend, no build step. Saves to `localStorage`.

## Use

Open `index.html` directly in a browser, or visit the GitHub Pages deployment.

## Stack

- HTML / CSS (custom, no framework)
- Vanilla JS, ES2022
- Inter + Sora + Cormorant Garamond + JetBrains Mono via Google Fonts

## Files

- `index.html` — markup, 5 tabs
- `styles.css` — all styling
- `data.js` — workout protocol (exercises, supplements, rules, phases)
- `app.js` — state, localStorage, chart, render

Reset everything from the Protocol tab.
